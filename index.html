GitHub needs a key so the site can save your content there. Here's how to get one:

1. Open GitHub → Settings → Developer settings → Fine-grained tokens
2. Click Generate new token
3. Name it anything, set expiry to No expiration
4. Under Repository access, choose Only select repositories → pick your repo
5. Under Permissions → Contents, set to Read and Write
6. Click Generate token and paste it below
